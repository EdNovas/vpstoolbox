## 脚本

最新版：

```
wget -N https://raw.githubusercontent.com/ednovas/vpstoolbox/main/ednovastool.sh && chmod +x ednovastool.sh && ./ednovastool.sh
```

稳定版：

```
bash <(curl -sL https://suo.yt/vSeL8AH)
```
